# Potato Leaf Disease Classification

**Advanced Computational Techniques for Big Imaging and Signal Data**

Multi-class image classification of potato leaves using **ResNet18 transfer learning** on the full **PlantVillage** dataset.

**Repository:** [github.com/aman-abeb/Classification-with-Deep-Learning](https://github.com/aman-abeb/Classification-with-Deep-Learning)  
**Live demo:** [classification-with-deep-learning.streamlit.app](https://classification-with-deep-learning.streamlit.app/)

## Classes

| Folder                  | Disease      |
| ----------------------- | ------------ |
| `Potato___Early_blight` | Early Blight |
| `Potato___Late_blight`  | Late Blight  |
| `Potato___healthy`      | Healthy      |
|                         |

| Extra                              | Section in notebook              | Output                                                        |

## Hyperparameters

| Parameter     | Value                                                   |
| ------------- | ------------------------------------------------------- |
| Model         | ResNet18 (frozen backbone)                              |
| Image size    | 224                                                     |
| Batch size    | 32                                                      |
| Epochs        | 15                                                      |
| Learning rate | 1e-3                                                    |
| Optimizer     | Adam (classifier head)                                  |
| Split         | 70% / 15% / 15%                                         |
| Seed          | 42                                                      |
| Augmentation  | HorizontalFlip, VerticalFlip, RandomRotate90, Normalize |

## Reproduce

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

1. Place potato folders under `PlantVillage/`
2. Open `potato_disease_classification.ipynb` → **Run All** (section 11 trains a baseline model for a few epochs)
3. Launch demo: `streamlit run app.py`

## Live demo

The Streamlit app is deployed at:

**https://classification-with-deep-learning.streamlit.app/**

Upload a leaf image or use the sample images to get a prediction with optional Grad-CAM explainability.
